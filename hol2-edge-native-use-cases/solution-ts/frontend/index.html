Welcome to Spin

The CNCF project Spin is a framework for building and running fast, secure, and composable serverless applications using WebAssembly.

About Spin

  • Build serverless applications with WebAssembly
  • Deploy lightweight microservices
  • Write in your favorite language (Rust, Go, Python, JavaScript, and more)
  • Run anywhere - cloud, edge, or on-premises
